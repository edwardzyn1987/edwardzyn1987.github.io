# 诗词展示网站设计文档

## 概述

一个基于 Astro 的古体诗词个人展示网站，采用水墨古典视觉风格，展示 50+ 首古体诗词，支持分类浏览、搜索、朗诵播放和读者留言。

## 技术栈

- **框架**：Astro（静态站点生成器）
- **内容管理**：Markdown 文件 + Astro Content Collections
- **样式**：CSS（自定义，不依赖 UI 框架）
- **音频**：HTML5 Audio API
- **搜索**：客户端搜索（Fuse.js 或类似轻量方案）
- **评论**：Giscus（基于 GitHub Discussions，无需后端）
- **部署**：GitHub Pages 或 Vercel（静态托管）
- **字体**：Ma Shan Zheng（毛笔标题）、Noto Serif SC（正文宋体）、ZCOOL XiaoWei（副标题）

## 视觉风格：水墨古典

- **底色**：宣纸暖色渐变（`#f5f0e8` → `#e8dcc8`）
- **主色调**：古铜褐 `#8b7355`
- **文字色**：深墨 `#2c2c2c`（标题）、`#3c3c3c`（正文）、`#666`（次要）
- **标题字体**：Ma Shan Zheng，模拟毛笔书法
- **正文字体**：Noto Serif SC，清晰易读的宋体
- **装饰元素**：细线分隔、圆角诗词卡片、左侧褐色竖线标识
- **标签样式**：圆角胶囊形，褐色描边

## 页面结构

### 1. 首页

- 网站标题（毛笔字体居中）+ 副标题
- 精选诗词（3-5 首推荐作品）
- 最新发布
- 底部导航链接

### 2. 诗词列表页（核心页面）

**侧栏导航布局**：

- **左侧侧栏**（固定宽度 ~220px）：
  - 搜索框（圆角，placeholder "搜索诗词..."）
  - 分类导航：全部、五言绝句、七言绝句、五言律诗、七言律诗、词，每项带数量统计
  - 标签云：山水、思乡、咏物、春、夏、秋、冬等主题标签
- **右侧主区域**：
  - 顶部标题 + 总数统计
  - 诗词列表：每项显示标题、首句预览、分类标签
  - 点击进入详情页

### 3. 诗词详情页

**居中横排布局**：

- 面包屑导航（首页 / 分类 / 当前诗题）
- 毛笔字体标题（Ma Shan Zheng, 36px）
- 类别 + 创作时间
- 细线分隔
- 诗词正文（Noto Serif SC, 20px, 行高 2.4, 字间距 3px, 居中）
- 标签（分类 + 主题）
- 创作手记（可选，作者对该诗的创作背景说明）
- 底部：上一首 / 下一首 导航
- **朗诵播放按钮**：右下角浮动按钮（详见音频功能）

### 4. 关于作者页

- 作者介绍
- 写作理念 / 诗词观
- 联系方式或社交链接（可选）

### 5. 留言页

- Giscus 评论组件（基于 GitHub Discussions）
- 读者可以留言交流

## 音频朗诵功能

### 交互设计

- **浮动播放按钮**：固定在详情页右下角
- 圆形按钮，古铜褐底色 `#8b7355`，白色播放图标
- 播放时显示声波扩散动画（两层波纹）
- 按钮下方显示"聆听"文字标签
- 点击切换播放/暂停
- 按钮带阴影悬浮效果

### 技术实现

- 音频文件格式：MP3（兼容性最佳）
- 存放在 `public/audio/` 目录，文件名与诗词 slug 对应
- 使用 HTML5 `<audio>` 元素 + JavaScript 控制
- 没有音频文件的诗词不显示播放按钮
- 播放状态：图标从播放三角 → 暂停双竖线

### 数据结构

每首诗词的 Markdown frontmatter 增加音频字段：

```yaml
---
title: "秋夜怀远"
category: "七言绝句"
tags: ["思乡", "秋"]
date: 2024-10-15
audio: "qiu-ye-huai-yuan.mp3"  # 可选，无则不显示播放按钮
note: "秋夜独坐，月光如水..."   # 可选，创作手记
---
```

### 诗词 Markdown 示例

```markdown
---
title: "秋夜怀远"
category: "七言绝句"
tags: ["思乡", "秋"]
date: 2024-10-15
audio: "qiu-ye-huai-yuan.mp3"
note: "秋夜独坐，月光如水，忽忆远方故人。遂提笔写下此诗，聊寄思念之情。"
featured: true
---

露冷风清夜未央，
孤灯独坐忆潇湘。
遥知千里同明月，
不尽相思入梦长。
```

## 搜索功能

- 使用 Fuse.js 实现客户端模糊搜索
- 构建时生成搜索索引 JSON（标题 + 正文 + 标签）
- 搜索框实时过滤，无需服务器
- 搜索结果高亮匹配关键词

## 评论系统

- 使用 Giscus（https://giscus.app）
- 基于 GitHub Discussions，无需自建后端
- 支持 Markdown 格式回复
- 可放在留言页作为统一留言板，也可考虑每首诗下方独立评论区

## 响应式设计

- **桌面端**（> 768px）：侧栏 + 主区域双栏布局
- **移动端**（≤ 768px）：侧栏折叠为顶部下拉菜单，单栏布局
- 浮动播放按钮在移动端适当缩小

## 部署方案

- **推荐**：GitHub Pages（免费、与 Giscus 天然配合）
- 备选：Vercel / Netlify
- 自定义域名可选
- Astro 构建输出纯静态 HTML，无需服务器
